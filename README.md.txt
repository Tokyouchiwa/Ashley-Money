# Ashley Money

Bienvenue sur le dépôt GitHub du site web **Ashley Money**. Ce site est conçu pour fournir des informations financières et des conseils en investissement, ainsi qu'une application pour aider les étudiants à investir dès le plus jeune âge dans des projets simples et efficaces.

## Aperçu

Ashley Money est un site web qui offre des conseils financiers et des services d'investissement. Notre objectif est d'aider les jeunes à développer de bonnes habitudes financières et à investir intelligemment dès le plus jeune âge.

## Fonctionnalités

- Conseils personnalisés pour les investissements
- Tutoriels et ressources éducatives
- Suivi des performances des investissements
- Recommandations de projets adaptés aux jeunes investisseurs

## Installation

Pour installer et exécuter ce projet localement, suis ces étapes :

1. Clone le dépôt :
    ```bash
    git clone https://github.com/<ton_nom_utilisateur>/ashley-money.git
    ```

2. Accède au répertoire du projet :
    ```bash
    cd ashley-money
    ```

3. Ouvre le fichier `index.html` dans ton navigateur web.

## Utilisation

- **À propos** : Informations sur le site et son objectif.
- **Services** : Description des services financiers offerts.
- **Application** : Présentation de l'application pour les étudiants.
- **Contact** : Informations de contact.
